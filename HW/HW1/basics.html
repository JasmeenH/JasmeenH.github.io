Java Basics

Java programs are written using a clear and structured syntax. Every Java program starts with a class, and execution begins from the main() method.

Java uses curly braces {} to define blocks of code and semicolons ; to end statements.

Variables and Data Types

Java uses variables to store data. Each variable must have a data type, which defines what kind of value it can hold.

Common data types include:

  • int
  • double
  • char
  • boolean
  • String
  • Arrays

Control statements

Java provides control statements to manage program flow:

  • if/else
  • for loop
  • while loop
  • switch
